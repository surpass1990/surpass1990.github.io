main.py

入口函数

python3 main.py：[option] {sec_name} {service|action} {action}
option: [必须]
    jenkins: 配置jenkins
        sec_name: 必须，config.ini中node名称
        action: {add | del}非必须，添加删除jenkins视图，任务列表, 默认值 add
        eg: python3 main.py jenkins template

    docker: 操作容器
        sec_name: 必须，config.ini中node名称
        action: {add | del} 非必须，添加、删除容器, 默认值 add
        eg: python3 main.py docker template del

    env: 构建容器中war、jar运行环境，配置jdk、zk、nginx等
        sec_name: 必须，config.ini中node名称
        python3 main.py env template

    build: 构建war、jar资源
        sec_name: 必须，config.ini中node名称
        service_name: 允许为空串，但是必须存在占位符
        eg: python3 main.py env template user
        eg: python3 main.py env template ""

    run: 运行容器中服务
        sec_name: 必须，config.ini中node名称
        service_name: 允许为空串，但是必须存在占位符
        action: {start|restart|stop} 非必须, 默认值 start
        eg: python3 main.py run template user start

其他模块如config_handler.py、container_handler.py等作用见脚本中注释
